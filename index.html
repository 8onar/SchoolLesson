HTML (HyperText Markup Language) - to standaryzowany język znaczników do tworzenia stron internetowych. Służy do określania struktury i zawartości strony za pomocą różnych elementów i tagów. HTML pozwala opisywać, jak zawartość strony powinna быть zorganizowana i wyświetlana w przeglądarce. Tagi HTML są używane do wskazywania różnych rodzajów treści, takich jak tekst, obrazy, odnośniki i inne elementy. Przeglądarki interpretują kod HTML i wyświetlają stronę zgodnie z określoną strukturą.

Przykład tytułu i akapitu:

                        <!DOCTYPE html>
                        <html lang="pl">
                        <head>
                            <meta charset="UTF-8">
                            <meta name="viewport" content="width=device-width, initial-scale=1.0">
                            <title>Przykład</title>
                        </head>
                        <body>
                            <h1>Tytuł</h1>
                            <p>To jest przykładowy paragraf.</p>
                        </body>
                        </html>

Przykład listy:
